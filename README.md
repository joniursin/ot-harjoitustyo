# Pacman

Sovelluksen tarkoituksena on luoda Pacman peli, joka toimii pääpiirteiltää samalla tavalla kuin alkuperäinen. Peli osaa pitää kirjaa siinä saavutetuista high scoreista ja tallentaa ne ylös.
